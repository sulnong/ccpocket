Built for mobile coding agents

Use Codex when you are away from your desk

CC Pocket is made for developers who want to keep a Codex or Claude Code session moving without staying in front of a desktop terminal. The agent runs on your own machine, while the phone app gives you the controls that matter on the go.

Start and resume Codex sessions

Self-hosted workflow

Your code stays on your machine

CC Pocket does not replace Codex or Claude Code. It adds a mobile client and a small Bridge Server that connects your phone to the coding tools already installed on your Mac or Linux machine.

1

Install Codex or Claude Code locally

Use the same coding-agent setup you already trust on your development machine.

2

Run the CC Pocket Bridge Server

Bridge exposes a WebSocket endpoint and prints a QR code for pairing the mobile app.

3

Connect from phone, tablet, or Mac

Use CC Pocket on iPhone, iPad, Android, or macOS to control sessions from another room or over Tailscale.

Questions

Codex from phone FAQ

Can I run Codex from my phone?

Yes. CC Pocket connects your phone to a self-hosted Bridge Server on the machine where Codex is installed, so you can start, resume, and monitor Codex sessions from iPhone, iPad, or Android.

Does CC Pocket send my code to a hosted service?

No. CC Pocket is designed around a self-hosted Bridge Server. Your coding-agent session runs on your own Mac or Linux machine.

Is CC Pocket free?

Yes. CC Pocket is free to use. You still need your own Codex or Claude Code setup for the agent session itself.
