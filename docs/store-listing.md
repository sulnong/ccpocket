## Full Description

### English

AI coding agents can now implement complete features. You still need to guide the work, approve actions, answer questions, and review the results. CC Pocket brings that workflow to your phone, tablet, and Mac.

CC Pocket is a mobile and desktop companion for OpenAI Codex and Claude Code. Your agents run through a self-hosted Bridge Server on your own Mac or Linux machine, while you start sessions, make decisions, inspect files, and manage changes from iPhone, iPad, Android, or macOS.

This app is not affiliated with, endorsed by, or associated with Anthropic or OpenAI.

**Key Features**

- Self-hosted Bridge Server on your own Mac or Linux machine
- Start Codex or Claude Code sessions from the app
- Resume sessions started from Codex CLI, the Codex app, or Claude Code
- Approve commands, file edits, MCP requests, and agent questions
- Use iPad / macOS workspace layouts for chat, Git, Explorer, images, and screenshots
- Browse project files with Explorer
- Review syntax-highlighted diffs and image diffs
- Stage changes, generate commit messages, commit, push, or revert
- Recover missed streaming updates, queue outgoing messages while offline, and resend after reconnecting
- Write prompts with Markdown, completions, voice input, and image attachments
- Connect via QR code, mDNS, saved machines, manual URL, or Tailscale
- Optional Supporter purchases for ongoing development

**How It Works**

1. Install the Bridge Server: `npx @gotokens/bridge@latest`
2. Open ccpocket and scan the QR code or enter the URL
3. Start coding from your phone, tablet, or Mac

**Requirements**

- A Mac or Linux machine with OpenAI Codex CLI, Claude Code CLI, or both installed
- Network connectivity between your device and the server (local network or Tailscale recommended)

Source code available at https://github.com/K9i-0/ccpocket
