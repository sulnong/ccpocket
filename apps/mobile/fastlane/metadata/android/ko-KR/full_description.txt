AI 코딩 에이전트는 이제 기능 하나를 처음부터 끝까지 구현할 수 있습니다. 개발자의 일은 지시하고, 승인하고, 답하고, 리뷰하는 판단입니다. CC Pocket은 그 제어 화면을 휴대폰, 태블릿, Mac으로 가져옵니다.

CC Pocket은 OpenAI Codex / Claude Code를 위한 모바일 및 데스크톱 클라이언트입니다. 에이전트는 사용자의 Mac / Linux 머신에 셀프 호스팅한 Bridge Server에서 실행되고, 사용자는 Android, iPhone, iPad, macOS에서 세션을 시작하고, 결정을 내리고, 파일을 확인하고, 변경 사항을 반영할 수 있습니다.

이 앱은 Anthropic 또는 OpenAI와 제휴하거나, 승인 또는 보증을 받은 제품이 아닙니다.

<b>셀프 호스팅 중심 설계</b>
Bridge Server는 사용자의 머신에서 실행됩니다. 프로젝트, shell 접근, git 저장소, 에이전트 세션은 사용자의 관리 아래에 남습니다. 텔레메트리나 추적은 없습니다.

<b>세션 시작과 이어가기</b>
앱에서 Codex / Claude Code 세션을 시작하고, 프로젝트와 권한 모드를 선택한 뒤 몇 초 안에 작업을 시작하세요. Codex CLI, Codex App, Claude Code에서 만든 세션도 Recent Sessions에 표시되어 다른 기기에서 이어갈 수 있습니다.

<b>모바일에 맞춘 승인</b>
터치에 최적화된 UI로 명령 실행, 파일 편집, MCP 요청, 에이전트 질문을 검토하세요. 여러 세션에 승인 대기가 있어도 세션 목록에서 한 번에 확인하고 처리할 수 있습니다.

<b>큰 화면 워크스페이스</b>
iPad와 macOS에서는 화면 공간에 맞춰 워크스페이스 레이아웃을 사용합니다. 채팅, 세션 목록, Git, Explorer, 이미지, 스크린샷을 나란히 둘 수 있습니다.

<b>프로젝트 파일 탐색</b>
Explorer로 휴대폰이나 태블릿에서 프로젝트 파일을 둘러보세요. 노트북을 열지 않고도 에이전트가 다루는 내용을 확인할 수 있습니다.

<b>Git 리뷰와 작업</b>
구문 강조 diff와 이미지 diff를 확인하고, 파일 통계를 살펴보며, 변경 사항 stage, 커밋 메시지 생성, commit, push, revert를 앱에서 실행할 수 있습니다.

<b>불안정한 네트워크 대응</b>
실제 모바일 네트워크를 고려했습니다. 누락된 메시지 델타를 복구하고, 오프라인 메시지를 pending으로 유지하며, 재연결 후 자동으로 다시 전송할 수 있습니다.

<b>풍부한 컨텍스트</b>
Markdown, 자동완성, 음성 입력, 이미지 첨부로 프롬프트를 작성하세요. MCP 도구가 촬영한 이미지를 보고, 시각적 컨텍스트가 필요할 때 Mac 스크린샷을 요청할 수 있습니다.

<b>유연한 연결</b>
저장된 머신, QR 코드, mDNS 자동 검색, 수동 URL로 연결하세요. 집이나 사무실 밖에서는 Tailscale을 권장하며, 필요하면 SSH로 원격 서버를 관리할 수 있습니다.

<b>선택적 Supporter</b>
CC Pocket은 계속 무료로 사용할 수 있습니다. 지속적인 개발을 후원하고 싶다면 선택적 일회성 또는 월간 Supporter 구매를 이용할 수 있습니다.

<b>사용 방법</b>
1. Bridge Server 설치: npx @gotokens/bridge@latest
2. 앱에서 QR 코드를 스캔하거나 URL 입력
3. 휴대폰, 태블릿, Mac에서 코딩 시작

<b>필수 조건</b>
• OpenAI Codex CLI 및/또는 Claude Code CLI가 설치된 Mac / Linux 머신
• 기기와 서버 사이의 네트워크 연결(로컬 네트워크 또는 Tailscale 권장)
