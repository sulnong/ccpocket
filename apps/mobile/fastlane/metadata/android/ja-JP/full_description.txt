AIコーディングエージェントが、機能まるごと実装できる時代。開発者に必要なのは、指示・承認・回答・レビューという判断です。CC Pocket は、その操作画面をスマホ、タブレット、Mac に持ち出せるアプリです。

CC Pocket は、OpenAI Codex / Claude Code のモバイル・デスクトップクライアントです。エージェントは自分の Mac / Linux マシン上のセルフホスト Bridge Server で動き、Android、iPhone、iPad、macOS からセッション開始、承認、ファイル確認、変更レビュー、コミットまで行えます。

本アプリはAnthropic・OpenAIとは無関係であり、両社の公認・推薦を受けたものではありません。

<b>セルフホスト前提</b>
Bridge Server は自分のマシンで実行します。プロジェクト、シェル、git リポジトリ、エージェントの会話は自分の管理下に残ります。テレメトリやトラッキングはありません。

<b>セッション開始と再開</b>
アプリから Codex / Claude Code セッションを開始できます。プロジェクトとパーミッションモードを選んで、すぐに作業開始。Codex CLI、Codex App、Claude Code で作ったセッションも Recent Sessions に表示され、別端末から続きを開けます。

<b>モバイル向け承認 UI</b>
コマンド実行、ファイル編集、MCP リクエスト、エージェントの質問をタッチ操作で確認。複数セッションに承認待ちがある場合も、セッション一覧からまとめて判断できます。

<b>大画面ワークスペース</b>
iPad / macOS では画面幅に合わせてワークスペースレイアウトに切り替わります。チャット、セッション一覧、Git、Explorer、画像、スクリーンショットを横に並べて扱えます。

<b>Explorer でファイル確認</b>
プロジェクト内のファイルをスマホやタブレットから閲覧できます。エージェントが触っている内容を、PC を開かずに確認できます。

<b>Git レビューと操作</b>
シンタックスハイライト付き diff と画像 diff を確認し、ファイル統計、stage、コミットメッセージ生成、commit、push、revert までアプリから操作できます。

<b>通信が不安定でも続けられる</b>
実際のモバイル回線を考慮しています。取りこぼしたメッセージ差分の復元、オフライン中の pending 送信、オンライン復帰後の自動再送に対応しています。

<b>リッチなコンテキスト</b>
Markdown、補完、音声入力、画像添付でプロンプトを作成。MCP ツールが撮影した画像の閲覧や、Mac のスクリーンショット取得にも対応しています。

<b>柔軟な接続</b>
保存済みマシン、QR コード、mDNS 自動検出、手動 URL で接続できます。外出先からは Tailscale の利用がおすすめです。必要に応じて SSH 経由のリモート管理も使えます。

<b>任意の Supporter</b>
CC Pocket は無料で使えます。継続開発を応援したい方向けに、任意の単発・月額 Supporter 購入を用意しています。

<b>セットアップ方法</b>
1. Bridge Server をインストール: npx @gotokens/bridge@latest
2. アプリから QR コードをスキャン、または URL を入力して接続
3. スマホ、タブレット、Mac からコーディング開始

<b>利用要件</b>
• Codex CLI または Claude Code CLI がインストール済みの Mac / Linux マシン
• 端末とサーバー間のネットワーク接続（ローカルネットワークまたは Tailscale 推奨）
