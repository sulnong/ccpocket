AI coding agents can now write entire features autonomously. Your job? Approve, answer, review. That doesn't need a laptop — it needs your phone.

CC Pocket is a mobile client for OpenAI Codex and Claude Code. Start Codex sessions from Android, iPhone, iPad, or Mac, make decisions on the go, and keep the agent running on your own machine.

This app is not affiliated with, endorsed by, or associated with Anthropic or OpenAI.

<b>START SESSIONS FROM YOUR PHONE</b>
Launch new Codex or Claude Code sessions directly from your phone — no need to open a laptop first. Pick a project, set a permission mode, and start coding in seconds.

<b>BATCH APPROVAL AT A GLANCE</b>
Multiple agents running? See all pending approvals across sessions in one list. Review and approve file edits, command executions, and tool calls without switching context.

<b>MOBILE-OPTIMIZED UI</b>
Built for touch. Answer agent questions, approve tools, and provide feedback through a UI designed for mobile — not a shrunken desktop terminal.

<b>RICH PROMPTS</b>
Write with Markdown formatting and auto-completing bullet lists. Attach images from your clipboard or photo gallery to give visual context to your agent.

<b>ORGANIZE YOUR SESSIONS</b>
Name your sessions and browse by project. Find past conversations instantly with search and project-based filtering.

<b>REVIEW DIFFS</b>
See every code change at once with syntax-highlighted diffs. Review file statistics, select specific change blocks, and send feedback.

<b>REAL-TIME STREAMING</b>
Watch your agent think and code live. View inline diffs, thinking blocks, and structured plans as they stream in.

<b>PUSH NOTIFICATIONS</b>
Get notified instantly when your agent needs approval or completes a task. Never miss a permission request again.

<b>OPTIONAL SUPPORTER</b>
CC Pocket remains fully usable for free. If you want to support ongoing development, the app also includes optional one-time and monthly Supporter purchases, a Support screen with purchase history summaries, and alternate app icon perks for monthly supporters.

<b>FLEXIBLE CONNECTIONS</b>
Connect to your self-hosted Bridge Server via saved machines, QR code, mDNS auto-discovery, or manual URL. Optionally manage remote servers over SSH.

<b>PRIVACY FIRST</b>
Your code and conversations stay on your own server. No telemetry, no tracking. CC Pocket collects no personal data.

<b>HOW IT WORKS</b>
1. Install the Bridge Server: npx @ccpocket/bridge@latest
2. Scan the QR code or enter the URL in the app
3. Start coding from your phone

That's it — you're ready to go in under 5 minutes.

<b>REQUIREMENTS</b>
• A Mac or Linux machine with OpenAI Codex or Claude Code CLI installed
• Network connectivity between your phone and the server (local network or Tailscale recommended)
