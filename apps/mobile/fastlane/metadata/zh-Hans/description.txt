AI 编程代理已经能够实现完整功能。开发者需要做的是指导、审批、回答和审查。CC Pocket 把这个控制界面带到你的手机、平板和 Mac 上。

CC Pocket 是 OpenAI Codex / Claude Code 的移动与桌面客户端。代理通过你自己的 Mac / Linux 主机上的自托管 Bridge Server 运行，你可以从 iPhone、iPad、Android 或 macOS 启动会话、做出决策、查看文件并交付变更。

本应用与 Anthropic、OpenAI 无关，未获得两家公司的认可或推荐。

为自托管而设计
Bridge Server 运行在你自己的机器上。项目、shell、git 仓库和代理会话都留在你的控制之下。没有遥测，也没有跟踪。

启动并继续会话
直接从应用启动 Codex / Claude Code 会话，选择项目和权限模式，几秒钟即可开始。通过 Codex CLI、Codex App 或 Claude Code 创建的会话也会出现在 Recent Sessions 中，方便在另一台设备上继续。

面向移动端的审批
用适合触屏的界面审查命令执行、文件编辑、MCP 请求和代理问题。当多个会话需要处理时，可以在会话列表中统一查看并审批。

大屏工作区
iPad 和 macOS 会根据屏幕空间使用自适应工作区布局。聊天、会话列表、Git、Explorer、图片和截图可以并排显示。

浏览项目文件
使用 Explorer 从手机或平板浏览项目文件。无需打开笔记本，也能确认代理正在处理的内容。

Git 审查与操作
查看带语法高亮的 diff 和图片 diff，检查文件统计，stage 变更，生成提交信息，commit、push 或 revert。

适应不稳定网络
CC Pocket 面向真实移动网络设计。它可以恢复遗漏的消息增量，在离线时保留 pending 消息，并在重新连接后自动重发。

丰富的上下文
使用 Markdown、补全、语音输入和图片附件编写提示。查看 MCP 工具拍摄的图片，并在需要视觉上下文时请求 Mac 截图。

灵活连接
通过已保存机器、二维码、mDNS 自动发现或手动 URL 连接。离开家或办公室时推荐使用 Tailscale，也可以选择通过 SSH 管理远程服务器。

可选 Supporter
CC Pocket 可以完整免费使用。如果你愿意支持持续开发，也可以选择单次或月度 Supporter 购买。

使用方法
1. 安装 Bridge Server：npx @gotokens/bridge@latest
2. 在应用中扫描二维码或输入 URL
3. 从手机、平板或 Mac 开始编程

使用要求
- 已安装 OpenAI Codex CLI 和/或 Claude Code CLI 的 Mac / Linux 主机
- 设备与服务器之间的网络连接（推荐局域网或 Tailscale）
