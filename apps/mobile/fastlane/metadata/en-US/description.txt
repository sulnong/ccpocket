AI coding agents can now implement complete features. You still need to guide the work, approve actions, answer questions, and review the results. CC Pocket brings that workflow to your phone, tablet, and Mac.

CC Pocket is a mobile and desktop companion for OpenAI Codex and Claude Code. Your agents run through a self-hosted Bridge Server on your own Mac or Linux machine, while you start sessions, make decisions, inspect files, and manage changes from iPhone, iPad, Android, or macOS.

This app is not affiliated with, endorsed by, or associated with Anthropic or OpenAI.

SELF-HOSTED BY DESIGN
Run the Bridge Server on your own machine. Your projects, shell access, git repository, and agent sessions stay under your control, with no telemetry or tracking.

START AND RESUME SESSIONS
Start new Codex or Claude Code sessions from the app. Pick a project, choose a permission mode, and begin in seconds. Sessions started from Codex CLI, the Codex app, or Claude Code also appear in Recent Sessions so you can pick them up from another device.

APPROVALS BUILT FOR MOBILE
Review command execution requests, file edits, MCP requests, and agent questions in a touch-first UI. When multiple sessions need attention, the session list shows pending approvals across them, so you do not have to jump between terminals.

WORKSPACE ON LARGER SCREENS
On iPad and macOS, an adaptive workspace layout lets chat, session list, Git, Explorer, images, and screenshots sit side by side when space allows.

EXPLORE YOUR PROJECT
Use Explorer to browse project files from your phone or tablet. See what the agent is changing without opening a laptop.

GIT REVIEW AND ACTIONS
Inspect syntax-highlighted diffs and image diffs, review file statistics, then stage changes, generate commit messages, commit, push, or revert from the app.

RESILIENT MOBILE WORKFLOW
Built for spotty mobile networks. CC Pocket can recover missed streaming updates, keep outgoing messages queued while offline, and send them after reconnecting.

RICH CONTEXT
Write prompts with Markdown, completions, voice input, and image attachments. View images captured by MCP tools, or request screenshots from your Mac when visual context matters.

FLEXIBLE CONNECTIONS
Connect using saved machines, QR codes, mDNS auto-discovery, or a manual URL. Use Tailscale for secure access away from home or the office, and optionally manage remote servers over SSH.

OPTIONAL SUPPORTER
CC Pocket remains fully usable for free. Optional one-time and monthly Supporter purchases are available if you want to support ongoing development.

HOW IT WORKS
1. Install the Bridge Server: npx @gotokens/bridge@latest
2. Scan the QR code or enter the URL in the app
3. Start coding from your phone, tablet, or Mac

REQUIREMENTS
- A Mac or Linux machine with OpenAI Codex CLI, Claude Code CLI, or both installed
- Network connectivity between your device and the server (local network or Tailscale recommended)
